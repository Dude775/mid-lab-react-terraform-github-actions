# Mid Lab - React Deployment with Terraform and GitHub Actions

This project deploys a React application to AWS using Terraform and GitHub Actions.

## Main Goal

Build a production-like deployment flow:

- Terraform infrastructure
- S3 static website hosting
- CloudFront distribution
- GitHub Actions CI/CD
- AWS authentication with OIDC
- CloudFront cache invalidation after deployment

- app/
- terraform/
- .github/workflows/
- images/
- architecture.drawio
- README.md
